# Assets

This folder keeps site resources out of the root page directory.

- `images/seasons/`: images from the original Dreams of Seasons page.
- `images/fashionlab/`: FashionLab cover and experiment placeholder images.
- `images/home/`: future homepage-specific cover images.
- `audio/seasons/`: audio from the original Dreams of Seasons page.
- `styles/`: page stylesheets.
- `fonts/`: optional future webfont files.
- `videos/`: future video files.
